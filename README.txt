Thinkific Uploader allows you migrate or share your Thinkific content

== Description ==

With Thinkific’s online learning plugin for WordPress, course creators can import WordPress content right into their Thinkific courses — plus, easily insert custom links (including additional prices) onto their Wordpress pages.

Thinkific makes building beautiful online courses simple. Whether you’re educating 10 students or 10 million, feel confident that you’ve got the easiest technology and the best support in the business.

Thinkific is used by more than 20,000 course creators supporting millions of students. Some of the features they love the most include:

* Instant Access to Funds
* Built-in SSL
* Easy Drag & Drop Course and Site Builder
* Course pricing options include Free, One Time Payment, Subscriptions (for Memberships), and Payment Plans
* Coupons, Promotions, and Completion Certificates
* Marketing & Email integrations including MailChimp, ConvertKit, and AWeber
* Additional integration options including Zapier, Webhooks, and API

== Installation ==

This section describes how to install the plugin and get it working.

1. Activate plugin
2. Add your API key & subdomain in the settings page
3. Click on the Save Changes button
